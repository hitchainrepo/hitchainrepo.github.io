创建 hit 配置

第一次使用 hit 时需要创建 hit 配置：hit cfg create your-password-here 


$> hit cfg create password
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
Please write down the mnemonic words(for recover your password)!!! :
alert maze mimic half fat depth motor trigger tape that ribbon also
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
Hit config is updated.
创建 hit 配置时，需要密码，并且生成一个密码助记词，可用于密码找回。

hit 密码恢复

通过助记词恢复密码。

  • hit 密码恢复命令

    • hit cfg recover password

    hit 帐户维护

    hit 可以维护多个帐户，并且可以随意切换到指定的名称的帐户上操作。 该帐户主要作用是用于与智能合约的交互。

  • hit 帐户相关命令

    • hit cfg account [add|remove] name [priKey] [pubKey] password
    hit cfg account set name

  • 创建帐户

    • hit cfg account add name [priKey] [pubKey] password
    参数说明：
    name：[必须]，帐户的名称，任意英文字母及数字
    priKey，pubKey：[可选]，添加已有的帐户时需要提供帐户私钥及帐户地址
    password：[必须]， 创建hit配置时的密码

  • 删除帐户

    • hit cfg account remove name
    参数说明：
    name：[必须]，帐户的名称

  • 设置默认帐户

    • hit cfg account set name
    参数说明：
    name：[必须]，帐户的名称

  • 列出所有帐户信息

    • hit cfg account

    hit Token 操作

    hit token 命令可以对 HIT 及以太坊进行简单操作

  • hit Token操作

    • hit token help
    hit token readToken [accountAddress]
    hit token readHitToken [accountAddress]
    hit token readContactToken contractAddress functionName [accountAddress]
    hit token requestTestToken [accountAddress]

  • 查看帐户中的以太坊

    • hit token readToken [accountAddress]
    参数说明：
    accountAddress：[可选]，指定帐户或当前 hit 配置中的帐户

  • 查看帐户中的HIT

    • hit token readHitToken [accountAddress]
    参数说明：
    accountAddress：[可选]，指定帐户或当前 hit 配置中的帐户

  • 查看以太坊代币

    • hit token readContactToken contractAddress functionName [accountAddress]
    参数说明：
    contractAddress：[必须]，智能合约的地址
    functionName：[必须]，调用的方法，目前只支持一个 Address 类型参数，并且返回值为 Uint256
    accountAddress：[可选]，指定帐户或当前 hit 配置中的帐户

  • 请求获得测试以太坊

    • hit token requestTestToken [accountAddress]
    参数说明：
    accountAddress：[可选]，指定帐户或当前 hit 配置中的帐户
