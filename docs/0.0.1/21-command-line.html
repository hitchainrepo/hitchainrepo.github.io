hit 命令说明

创建 hit 配置

第一次使用 hit 时需要创建 hit 配置：hit cfg create your-password-here 


$> hit cfg create password
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
Please write down the mnemonic words(for recover your password)!!! :
alert maze mimic half fat depth motor trigger tape that ribbon also
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
Hit config is updated.
创建 hit 配置时，需要密码，并且生成一个密码助记词，可用于密码找回。

hit 帐户维护

hit 可以维护多个帐户，并且可以随意切换到指定的名称的帐户上操作。 该帐户主要作用是用于与智能合约的交互。

  • hit 帐户相关命令

    • hit cfg account [add|remove] name [priKey] [pubKey] password
    hit cfg account set name

  • 创建帐户

    • hit cfg account add name [priKey] [pubKey] password
    参数说明：
    name：[必须]，帐户的名称，任意英文字母及数字
    priKey，pubKey：[可选]，添加已有的帐户时需要提供帐户私钥及帐户地址
    password：[必须]， 创建hit配置时的密码

  • 删除帐户

    • hit cfg account remove name
    参数说明：
    name：[必须]，帐户的名称

  • 设置默认帐户

    • hit cfg account set name
    参数说明：
    name：[必须]，帐户的名称

  • 列出所有帐户信息

    • hit cfg account
