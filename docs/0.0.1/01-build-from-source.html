源码构建

从源码构建hit。

准备安装环境

采用 Java 语言，所有平台设置相同。

安装 JDK8+

安装 JDK 8 以上版本 JDK 下载

安装 Maven

安装 Maven 3.5.2 以上版本 Maven 下载

检出源码并安装

通过源码安装hit。

检出源码

从 Github 仓库中检出 hit-java 源码。 


git clone https://github.com/hitchainrepo/hit-java.git
git checkout -b hit-0.01-5.2.1.2 origin/hit-0.01-5.2.1.2
源码

Maven 构建 


cd hit-java
mvn clean install -DskipTests

hit

hit 编译完成后存放的目录，复制到用户的执行目录下，如：/bin, /usr/bin。或复制到指定目录，然后添加系统 PATH 指向该目录。 


cd org.eclipse.jgit.pgm/target
cp hit YOUR_BIN_DIR
