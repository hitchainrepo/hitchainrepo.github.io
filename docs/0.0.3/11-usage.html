hit 使用说明

hit 与 git 兼容

hit 在很大程度上兼容 git 命令，目前 hit 可以正常执行所有 git 命令。 使用方式如下： 


hit [git command options]

hit 快速入门

hit 个性功能使用。

JDK 环境

目前 hit 只提供 Java 版本，运行需要 JDK 8 以上版本的支持。 安装 JDK 8 以上版本JDK 下载

hit 设置

在启用 hit 功能前需要对 hit 进行配置。配置分4部分：帐户、密钥对、存储、合约入口。 初始尝试 hit 可以尝试使用如下步骤进行配置： 


hit cfg help
hit cfg create              your-password-here
hit cfg account    add acc1 your-password-here
hit cfg rsa        add rsa1 your-password-here
hit cfg storage    add sto1 121.40.127.45
hit cfg repository add rep1 https://121.40.127.45:1443

hit 创建仓库

仓库一个新的仓库 


mkdir hello-world
cd    hello-world
hit   init
echo  "hello world" > helloworld.txt
hit   commit -m "your comment"
hit   remote add origin hit://hello-world.git
hit   push
