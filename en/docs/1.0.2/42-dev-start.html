Introduce fo Modules

hit and two modules to the project:

  • org.hitchain.core hit core module
  • org.hitchain.core.test hit test module
  • org.hitchain.pgm hit cli module

    • Module Design

    In order to minimize the changes to the original JGit source code, hit mainly adopts reflection injection for function implantation. For the code modification part of JGit, use the specific identification/*add by hit*//*end of add*/, you can search the code identification of the whole project, you can know the part of the source code modification of JGit.
