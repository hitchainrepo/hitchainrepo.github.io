Ethereum TestNet

At present, the accelerator is connected to the test network by default, which is mainly convenient for users to test. https://ropsten.etherscan.io.

Get Ethereum Token

The current default is to write the smart contract through the accelerator account's test account, and later versions will provide an option for users to write the smart contract using their own account. The ETH of the Ethereum testnet is available via the following link: Get Ethereum Token. or from the command line: 


wget https://faucet.ropsten.be/donate/<your ethereum address>
