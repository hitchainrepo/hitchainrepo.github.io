hit Usage

hit is compatible with git

hit is largely compatible with Git commands, and currently hits can execute all Git commands correctly. The usage is as follows: 


hit [git command options]

JDK Environment

Currently, HIT is available only in Java and requires JDK 8 or higher support to run. Install JDK 8+JDK Download

hit Settings

Hit needs to be configured before it can be enabled. Configuration consists of four parts: account, key pair, storage, and contract entry. The initial try hit can be configured using the following steps: 


hit cfg create
hit token request-test
hit cfg contract   deploy main
hit contract add-repository --auto-rename

hit Create Repository

Create new repository 


mkdir hello-world
cd    hello-world
hit   init
echo  "hello world" > helloworld.txt
hit   commit -m "your comment"
hit   contract add-repository --auto-rename
hit   push
