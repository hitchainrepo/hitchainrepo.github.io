    hit 帐户维护

    hit 可以维护多个帐户，并且可以随意切换到指定的名称的帐户上操作。 该帐户主要作用是用于与智能合约的交互。

  • hit 帐户相关命令

    • hit cfg account [add|remove] name [priKey] [pubKey] password
    hit cfg account set name

  • 创建帐户

    • hit cfg account add name [priKey] [pubKey] password
    参数说明：
    name：[必须]，帐户的名称，任意英文字母及数字
    priKey，pubKey：[可选]，添加已有的帐户时需要提供帐户私钥及帐户地址
    password：[必须]， 创建hit配置时的密码

  • 删除帐户

    • hit cfg account remove name
    参数说明：
    name：[必须]，帐户的名称

  • 设置默认帐户

    • hit cfg account set name
    参数说明：
    name：[必须]，帐户的名称

  • 列出所有帐户信息

    • hit cfg account

    智能合约交易的 GAS 配置

    hit 可以维护多个智能合约交易的 GAS 配置，并且可以随意切换到指定的名称的智能合约交易的 GAS 配置上操作。智能合约交易时会使用到该 GAS 配置，包括合约部署的 GAS 及写入合约的 GAS。

  • 智能合约交易的 GAS 配置相关命令

    • hit cfg gas add name deployGas deployGwei writeGas writeGwei
    hit cfg gas remove name
    hit cfg gas set name

  • 创建智能合约交易的 GAS 配置

    • hit cfg gas add name deployGas deployGwei writeGas writeGwei
    参数说明：
    name：[必须]，智能合约交易的 GAS 配置的名称，任意英文字母及数字
    deployGas：[必须]， 部署合约 GAS
    deployGwei：[必须]， 部署合约的 GWei
    writeGas：[必须]， 写合约的 GAS
    writeGwei：[必须]， 写合约的 GWei

  • 删除智能合约交易的 GAS 配置

    • hit cfg gas remove name
    参数说明：
    name：[必须]，智能合约交易的 GAS 配置的名称

  • 设置默认智能合约交易的 GAS 配置

    • hit cfg gas set name
    参数说明：
    name：[必须]，智能合约交易的 GAS 配置的名称

  • 列出所有智能合约交易的 GAS 配置信息

    • hit cfg gas
