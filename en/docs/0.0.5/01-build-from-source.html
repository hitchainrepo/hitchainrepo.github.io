Build From Source

Build hit From Source.

Prepare Installation Environment

Use Java, platform cross.

Install JDK8+

Install JDK 8 (build 1.8.0_191-b12) and above version. JDK Download

Install Maven

Install Maven 3.5.2 and above Maven Download

Check out source codes and install

Install hit by source codes.

Check out source codes.

Check out hit-java source codes from Github. 


git clone https://github.com/hitchainrepo/hit-java.git
git checkout -b hit-0.0.5-5.2.1.2 origin/hit-0.0.5-5.2.1.2
Source Codes

Maven Build 


cd hit-java
mvn clean install -DskipTests

hit

copy hit to exe dirs, such as: /bin or /usr/bin. Or you can copy to any directory and setting the PATH. 


cd org.eclipse.jgit.pgm/target
